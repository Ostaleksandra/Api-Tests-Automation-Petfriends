# PetFriends API Test Automation

Автоматизированные тесты для API сервиса **PetFriends**.

##  Описание проекта

Проект содержит Python-клиент для работы с API PetFriends и набор автоматических тестов, написанных с использованием **pytest**.

Тесты проверяют основные функции API: получение ключа авторизации, работу с питомцами (создание, изменение, удаление, добавление фото).

##  Проверяемый функционал

В проекте реализованы тесты для проверки:

* получение API-ключа (валидные и невалидные данные)
* получение списка питомцев
* добавление питомца
* добавление питомца без фотографии
* добавление фотографии питомца
* изменение данных питомца
* удаление питомца
* негативные сценарии (неверные данные)

##  Используемые технологии

* Python 3
* requests
* pytest
* python-dotenv

##  Настройка окружения

Создайте файл `settings.py` в корне проекта и добавьте данные пользователя:

```
valid_email=your_email
valid_password=your_password
```

##  Запуск тестов

Установите зависимости:

```
pip install requests pytest python-dotenv
```

Запустите тесты:

```
pytest -v tests.py
```

##  Цель проекта

Практика:
